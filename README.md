# PIBUPIBU Web Service

## 📌 프로젝트 개요
사용자 주문·결제 및 관리자 운영 기능을 포함한 웹 서비스 프로젝트입니다.
JSP 기반 MVC 구조로 구현하였으며, Oracle Database와 연동하여
실제 서비스 흐름을 고려한 기능을 개발하였습니다.

## 🛠 개발 환경
- OS : Windows
- Server : Apache Tomcat
- Database : Oracle Database
- Language : Java, JSP, HTML, CSS, JavaScript
- Tool : Eclipse, EditPlus, Git, SQL Developer

## 🔑 주요 기능
### 사용자
- 회원가입 / 로그인
- 상품 조회 및 주문
- 결제 및 쿠폰 적용
- 문의 등록

### 관리자
- 관리자 로그인
- 회원 관리
- 문의 답변
- 쿠폰 관리
